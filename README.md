Snake Game in C++

A simple Snake Game built using C++. This project runs in the terminal and was created to practice core C++ programming concepts and game development logic.

## Features

- Snake movement using keyboard controls
- Food generation
- Score tracking
- Collision detection
- Game Over screen

## Technologies Used

- C++
- Standard Library
- Console/Terminal

## Controls

- W - Move Up
- A - Move Left
- S - Move Down
- D - Move Right
- X - Exit Game
